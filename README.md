Opções de disponibilidade: 

Zona de disponibilidade: Separe fisicamente seus recursos em uma região do azure, até 3 zonas de infraestrutura para redundância.
Conjunto de dimensionamento de VMs: Distribua as VMs em zonas e domínios de falha em escala

Conjunto de disponibilidade: Distribua suas VMs automaticamente entre vários dominios de falha.
Executar com desconto do spot do azure: Azure Spot oferece a capacidade não usada do Azure a uma taxa com desconto, ao contrário dos preços pagos conforme o uso. As cargas de trabalho devem ser tolerantes à perda de infraestrutura, pois o Azure pode recuperar a capacidade para as cargas de trabalho pagas conforme o uso.
